# ninetails
A fun and simple little web browser

## Development
```bash
# Clone the repo
git clone https://github.com/MystPi/ninetails.git
cd ninetails

# Install the dependencies
npm install

# Start the app
npm start
```

## Building the app
```bash
# Clone the repo
git clone https://github.com/MystPi/ninetails.git
cd ninetails

# Install the dependencies
npm install

# Make the app
npm run make

# The built app should be located at `./out/make/{filetype}/{architecture}/{filename}.{filetype}`
# Eg. `./out/make/deb/arm64/ninetails_1.0.0_arm64.deb`
```
